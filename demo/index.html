Scroll-Incr

jQuery plugin for number increment when an element become visible

Follow @iam_e200

Intro

Scroll-Incr is a jQuery plugin for increment numbers on elements when they become visible on the view port.

Usage

Download or install via npm. 

npm i scroll-incr --save

Now, in the HTML:

<!DOCTYPE html>
<html>
<head>
  ...
  <script src="/jquery.min.js"></script>
  <script src="/scroll-incr.min.js"></script>
</head>
<body>
  <div class="scroll-incr" data-max="10"></div>
  <div class="scroll-incr" data-max="1000" data-increment="10"></div>
  <div class="scroll-incr" data-max="599" data-duration="8"></div>
  <div class="scroll-incr" data-min="100" data-max="10000"></div>
</body>
</html>

Make sure to include the jQuery library before include Scroll-Incr plugin.

Example

<div class="box">
  <p>Deserunt ipsum veniam quis.</p>
  <div class="scroll-incr" data-min="10" data-max="100" data-duration="5" data-steps="10">0</div>
</div>

<div class="box">
  <p>Aliquip sint labore est tempor et.</p>
  <div class="scroll-incr" data-max="604" data-steps="2">0</div>
</div>

<div class="box">
  <p>Consequat voluptate incididunt.</p>
  <div class="scroll-incr" data-max="1000" data-duration="2">0</div>
</div>

<div class="box">
  <p>Eiusmod commodo magna nulla ea.</p>
  <div class="scroll-incr" data-max="6028" data-steps="100">0</div>
</div>
